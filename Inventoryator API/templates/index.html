Inventoryator Rest Client API

/inventory/
GET
Returns a list of all inventories available as list of objects.
POST
Creates a new inventory.
Post body must be of form {"name": str, "description": str}
Data returned includes assigned database id.
/inventory/(int:id)/
GET
Returns inventory with given as object.
PUT
Updates the inventory with the given id.
Put body must be of form {"name": str, "description": str}, though only desired updates need to be included.
Data returned is the updated inventory data.
DELETE
Deletes the inventory with the given id from the database.
/inventory/(int:id)/items/
GET
Returns a list of items in the inventory with the given id, as a list of objects.
POST
Adds the provided item to the inventory with the given id.
Post body must be of the form {"name": str, "number": int}
Returns dadta for the item added, including its database id.
/items/(int:id)/
DELETE
Deletes the item with the given id from the database.
